# Single File Vulkan Pathtracing

Minimal pathtracer using Vulkan RayTracing

![](img/img.png)

# Environment

-   Vulkan SDK 1.2.162.0
-   GPU / Driver that support Vulkan Ray Tracing
-   C++17 or later

## Setup

See [vulkan-tutorial](http://vulkan-tutorial.com/)
